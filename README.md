# Password-Manager-App

### I want this application to do a few things.
- Create a master password for the app
- create login screen after password has been made
- store in a table the passwords and be able to name where they are for
- you should be able to create, edit and delete the different entries
- encrypt the passwords in file so only readable while activly viewing in app
- saves for future viewing


# How will I accomplish that
## Language: **Java**
- UI elements will be JavaFX
- Maven for dependencies
- jBCrypt for password hashing
- Jackson for JSON decription

- [ ] Create a vewing window
